c++的模运算

我们先抛开具体的开发语言不谈，先谈谈在数学里，什么叫做模运算。

模运算用mod来表示，对于a mod b可以表示为 a = qb + r,其中q表示商，b表示模数目且b!=0,那么余数r满足0<=|r|<|b|。

如果a和b都是自然数，那么r肯定是大于等于0且小于b的整数，如果a和b有一个是负数，那么r就不一定唯一。看下面这个例子：

（-1）% 256：-1=（0）x 256-1，余数是-1；-1=(-1) x 256+255,余数是255

从上面这个例子可以看出：两个余数的绝对值的和等于模数的绝对值。

那么问题来了，两个整数的模运算竟然有两个结果，那么我们到底如何做选择呢？
